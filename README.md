# Shopping Mart Management System

## Description
The Shopping Mart Management System is a comprehensive software solution designed to handle inventory management, billing, and sales tracking. Built using **Python, Streamlit, and MySQL**, it provides an intuitive interface for store owners and staff to manage products, generate invoices, and maintain sales records efficiently.

## Features
- **Inventory Management**: Add, update, delete, and track product stock.
- **Billing System**: Generate invoices and process transactions seamlessly.
- **Sales Tracking**: Maintain records of all sales for analysis and reporting.
- **User Authentication**: Secure login system for administrators and employees.
- **Database Integration**: Stores data in **MySQL** for efficient retrieval and updates.
- **Real-time Data Updates**: Ensures accurate stock and sales management.

## Technologies Used
- **Frontend**: Streamlit (Python-based UI framework)
- **Backend**: Python
- **Database**: MySQL

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/shopping-mart-management.git
   ```
2. Navigate to the project directory:
   ```sh
   cd shopping-mart-management
   ```
3. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
4. Set up the MySQL database and update connection details in the project.
5. Run the application:
   ```sh
   streamlit run app.py
   ```

## Usage
1. **Login**: Access the system using valid credentials.
2. **Manage Inventory**: Add, edit, or remove products from the inventory.
3. **Billing Process**: Generate invoices for customer purchases.
4. **Sales Tracking**: View and analyze transaction history.
